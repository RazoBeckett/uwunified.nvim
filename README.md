# Uwunified

- Theme inspired by the base16colorscheme named: "uwunicorn"
- Uwunify your Neovim now!

# Usage
- lazy
```lua
{
  "razobeckett/uwunified.com",
  config = function()
    vim.cmd.colorscheme("uwunified")
  end,
}
```
